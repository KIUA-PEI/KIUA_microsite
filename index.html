Recolha de KPIs e Geração de Dashboards

Recolher dados dos serviços da UA para posterior análise

Documentação

Contextualização

Atualmente é extremamente importante que qualquer organização tenha acesso a dados atualizados e em tempo real do que está a produzir.

Surge neste contexto o KIUA, um sistema cujo objectivo é recolher e armazenar dados-chave (KPIs) dos vários serviços da Universidade de Aveiro, para sua posterior apresentação em Dashboards e páginas Web que vão permitir a análise dos mesmos.

Objectivos

Principais objectivos a cumprir com este projeto

Sistema para monitorizar a UA

Desenvolvimento de um sistema que permita defenir e parametrizar KPIs (Key Performance Indicators) para serem recolhidas dos vários serviços da Universidade de Aveiro.

Bases de Dados & DashBoards

Desenvolvimento de bases de dados persistentes e robustas que permitam armazenar os dados recolhidos. Desenvolvimento de Dashboards que permitam a visualização dos dados em tempo real.

WebApp para consumo de dados

Desenvolvimento de um conjunto de páginas web que permitam o consumo dos dados armazenados e recolhidos em tempo real.

KPIs

KPIs a medir

Nº Resoluções DNS

Nº dispositivos ligados à Wifi

Nº Autenticações

Nº Autenticações falhadas

Nº Emails recebidos no servidor Exchange

Nº emails enviados

Módulos

Módulos que vão ser desenvolvidos

Módulo Kafka

Módulo que permite armazenar os dados recolhidos em tópicos especificos em em message broker Kafka para serem consumidos por outros módulos do projeto.

Base de dados

Base de dados para armazenar todos os dados resultantes do processo de recolha.

Dashboards

Dashboards de consumo dos dados recolhidos para apresentação dos mesmos nos mais diversos formatos e representações. .

Páginas Web

Conjunto de páginas web para consumo dos dados recolhidos em tempo real.

Frameworks e API

Framework para implementar hooks que permitem aceder aos dados dos diferentes serviços da UA. API REST registada no BUS de serviços da UA para consumo dos dados por aplicações terceiras.

Backoffice

Interface que permite a parametrização e especificação das KPIs a recolher.
